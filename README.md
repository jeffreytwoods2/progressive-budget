# progressive-budget

## Overview

A budget tracker that works even when your computer is offline. Save data locally, view changes in real time, and upload changes to the database once you're back online.

## Technologies:

<ul>
  <li>HTML/CSS/JS</li>
  <li>Webmanifest</li>
  <li>Service Tracker</li>
  <li>IndexedDB</li>
</ul>
